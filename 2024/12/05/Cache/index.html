Cache Access

  • If cache latency is 4 ns and mem latency is 40 ns. The cache miss latency is 44ns. \[ Miss Penalty (MP)= T_{request} + T_{access} + T_{return} \]

\[ Hit Ratio (HR) = \frac{hits}{hits + misses} \] \[ Miss Ratio (MR) = \frac{misses}{hits + misses} \] \[ Average Memory Access Time (AMAT) = T_{hit} + T_{MP} \times MR \]
