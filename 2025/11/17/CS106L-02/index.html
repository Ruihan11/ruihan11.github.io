第5讲：容器（Containers）📦

容器是 C++ 中组织数据的核心工具，提供类型安全、高性能的通用数据结构。

容器是什么

容器是封装了数据组织方式的模板类，分为两大类：

  • 序列容器：线性排列元素（如数组、双端队列）
  • 关联容器：按键组织元素（如字典、集合）

📌 所有 STL 容器都是模板（泛型），支持任意类型（只要满足接口要求）。

序列容器：线性结构

std::vector<T> —— 动态数组

  • 连续内存，缓存友好
  • 尾部操作高效：push_back / pop_back（摊还 O(1)）
  • 随机访问：v[i] 或 v.at(i)（后者越界抛异常）

⚠️ vector 没有 push_front —— 头部插入需整体搬移，极慢。

std::deque<T> —— 双端队列（读作 “deck”）

  • 分段连续内存（“数组的数组”）
  • 支持首尾高效操作：push_front / push_back / pop_front / pop_back
  • 仍支持 O(1) 随机访问（略慢于 vector）

✅ 适用场景：滑动窗口、任务队列等需双端操作的场景。

关联容器：按键索引

std::map<K, V> & std::set<K> —— 有序容器

  • 底层：红黑树（自平衡 BST）
  • 按键自动排序（中序遍历有序）
  • 查找/插入/删除：O(log n)
  • 要求键支持 operator<
  • std::set<T> ≈ map<T, void>：仅存唯一键，有序
  • 迭代遍历返回 std::pair<const K, V>：

std::unordered_map<K, V> & std::unordered_set<K> —— 无序哈希容器

  • 底层：哈希表 + 链地址法
  • 平均 O(1) 查找/插入/删除
  • 要求键可哈希（std::hash<K>）且可判等（operator==）
哈希关键概念：
术语 说明
哈希函数 hash(key) → size_t，应“打散”输入、抗冲突
负载因子 = 元素数 / 桶数；默认 >1.0 时触发 rehash
哈希冲突 不同键映射到同桶 → 链式线性查找（需判等）

✅ 内置支持：int、double、std::string
❌ 不支持：std::ifstream 等无默认哈希的类型

容器选型：空间 vs 时间

容器 随机访问 查找 插入 删除 有序 典型场景
vector ✅ O(1) O(n) 尾 O(1) 尾 O(1) ❌ 顺序数据、频繁遍历
deque ✅ O(1) O(n) 首尾 O(1) 首尾 O(1) ❌ 双端队列、滑动窗口
map/set ❌ O(log n) O(log n) O(log n) ✅ 需有序、范围查询
unordered_map/set ❌ O(1) avg O(1) avg O(1) avg ❌ 高频查找、无序可接受

🔑 经验法则：

  • 默认优先 unordered_map（快！）
  • 需有序遍历或 lower_bound → 用 map
  • 键无可比性但可哈希 → 只能用 unordered_map

💡 Stroustrup 名言：“Space is time.”
—— unordered_map 用更多内存换更快查找；vector 用紧凑内存换更慢查找。

其他常用容器（拓展）

容器 说明
std::array<T, N> 编译期定长数组，无动态开销
std::list<T> 双向链表，任意位置 O(1) 插删（缓存不友好）
std::multimap / multiset 允许重复键（有序）
std::unordered_multimap / ..._set 允许重复键（哈希）

第6讲：迭代器与指针

容器遍历的统一接口：迭代器

for-each 循环的底层实现原理：

核心抽象——“抓娃娃机”模型：

  • 容器 提供起止位置：begin()（第一个元素）、end()（尾后位置，永不指向有效元素）
  • 迭代器 支持四种基本操作：
    • *it：解引用，获取元素（若 it == end() 则未定义行为）
    • ++it：前移一步（推荐前缀，避免拷贝开销）
    • it == jt：相等性比较（用于终止判断）
    • it = container.begin()：初始化

💡 为什么用 ++it 而不是 it++？
后置自增 it++ 会构造一个旧值副本，对复杂迭代器对象（非 int）有额外开销；前缀 ++it 仅返回引用，永不更慢，有时更快（Bjarne 语）。

迭代器分类体系（按能力递增）

类型 能力 典型容器 算法兼容性示例
Input 单向只读 (*it, ++it) istream_iterator std::accumulate
Output 单向只写 (*it = val, ++it) ostream_iterator std::copy
Forward 多次单向遍历 unordered_map/set std::find
Bidirectional 双向移动 (++it, --it) map, set, list std::reverse
Random Access 随机跳转 (it + n, it[n], <) vector, deque std::sort, std::binary_search
