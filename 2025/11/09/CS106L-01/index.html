第2讲：类型和结构体

编译 vs 解释

C++ 是编译型语言：

  • 编译时：编译器扫描全部代码，检查错误
  • 运行时：执行已编译的程序

对比 Python（解释型）：

  • 运行时逐行解释和执行
  • 错误在代码执行到那行时才发现

关键差异： C++ 能在你运行代码前就捕捉很多错误！

静态类型系统

C++ 是静态类型语言——每个变量必须声明类型，声明后不能更改。

为什么要静态类型？

  • 效率更高（编译器能优化）
  • 代码更清晰、易理解
  • 错误更容易发现（编译时而非运行时）

结构体：打包多个数据

结构体让你把相关的变量组织在一起。

便捷工具

std::pair：两个元素的通用结构体

using 关键字：创建类型别名

auto 关键字：让编译器推断类型

L值 vs R值

  • R值：临时值，没有固定地址 - 通常在 = 右边
